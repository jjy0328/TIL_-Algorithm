
> Stack, Que
    컴퓨터 공학에서 가장 기본이 되는 자료구조
    자료를 표현하고 처리하는 한가지 방법


# 스택(Stack)

* 가장 먼저 들어온 것이 마지막에 나감 (FILO)
* ex) 택배 상하차 : 마지막에 들어온 데이터가 가장 먼저 처리
* 입구와 출구가 하나밖에 없는 상태로 표현 가능
* 자료구조에 가까운 이론


# 큐(Que)

* 가장 먼저 들어온 것이 가장 먼저 나감 (FIFO)
* ex) 은행창구 : 번호를 먼저 뽑으면 먼저 서비스를 받음
* 새치기 불가
